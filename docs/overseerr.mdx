# Getting New Media

Your server is preloaded with a selection of media to choose from. Having trouble finding something you like? 

**Overseerr** is a user-friendly request system that can automatically search for and grab the media you're after.

1. Request any available Movie or TV Show
2. Media will be automatically downloaded to your library
3. You can opt-in torecieve notifications when your media is ready to enjoy.
